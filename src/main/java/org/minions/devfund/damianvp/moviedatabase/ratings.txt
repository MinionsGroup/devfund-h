<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">Overwrite the current rating of a movie with this new rating. Again, assume that the name argument will definitely be a name that is currently in the database.</span></p>
<p></p>
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-family: arial, helvetica, sans-serif; color: #000000;"><em><span style="font-size: 12pt; background-color: transparent; font-weight: bold; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">String</span></em><span style="font-size: 12pt; background-color: transparent; font-weight: bold; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;"> getBestActor()</span></span></p>
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">Returns the name of the actor that has the best average rating for their movies.</span></p>
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span face="arial, helvetica, sans-serif" style="font-family: arial, helvetica, sans-serif;"><span style="white-space: pre-wrap;">&nbsp;</span></span></p>
<p><span color="#ff0000" style="color: #ff0000;">In the case of a tie, return any&nbsp;<strong><span style="text-decoration: underline;">one</span></strong> of the best actors.</span></p>
<p><em style="color: #000000; font-family: arial, helvetica, sans-serif; font-size: 1em;"><span style="font-size: 12pt; background-color: transparent; font-weight: bold; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">String</span></em><span style="font-size: 12pt; background-color: transparent; font-weight: bold; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;"> getBestMovie()<br></span><span style="font-family: arial, helvetica, sans-serif; color: #000000;">Returns the name of the movie with the highest rating.</span></p>
<p style="font-size: 16px;"><span color="#ff0000" style="color: #ff0000;">In the case of a tie, return any&nbsp;<strong><span style="text-decoration: underline;">one</span></strong>&nbsp;of the best movies.</span></p>
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"></p>
<p><span style="font-family: arial, helvetica, sans-serif; color: #000000;"><strong>Main method<span style="text-decoration: underline;"><br></span></strong>Finally, write a main method where:</span></p>
<p></p>
<ul>
<ul><ol style="margin-top: 0pt; margin-bottom: 0pt;">
<li dir="ltr" style="list-style-type: decimal; font-size: 12pt; font-family: Calibri; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline;">
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">You create a new instance of a movieDatabase.</span></p>
</li>
<li dir="ltr" style="list-style-type: decimal; font-size: 12pt; font-family: Calibri; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline;">
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">Add all the movies in the file movies.txt.</span></p>
</li>
<li dir="ltr" style="list-style-type: decimal; font-size: 12pt; font-family: Calibri; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline;">
<p dir="ltr" style="line-height: 1.2; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: arial, helvetica, sans-serif; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline; white-space: pre-wrap;">Go through the ratings of the movies in the file ratings.txt and add the ratings for the movies.</span></p>
</li>
<li dir="ltr" style="list-style-type: decimal; font-size: 12pt; font-family: Calibri; color: #000000; background-color: transparent; font-variant-ligatures: normal; font-variant-east-asian: normal; font-variant-position: normal; vertical-align: baseline;">
